## 📌 About

This is my personal portfolio website designed to feel like an **interactive 3D experience** rather than a traditional static page. Every animation, effect, and interaction is hand-coded from scratch using vanilla web technologies.

Built during **Day 7 & 8** of my **#100DaysOfCode** journey — started January 2026.

---

## 🎯 Features

- 🌌 **Fullscreen 3D Canvas Background** — Floating geometric shapes (triangles, circles, squares) + animated particle field built with the Canvas API
- 🖱️ **Custom Neon Cursor** — Instant-follow dot with a smooth trailing ring effect
- 🎭 **Mouse Parallax** — Hero section responds to mouse movement with a 3D perspective tilt
- 🃏 **3D Card Tilt** — About & Project cards rotate in 3D based on mouse position
- 🪟 **Glassmorphism UI** — Frosted glass cards with neon blue/purple glow accents
- 🌙 **Dark / Light Mode Toggle** — Full theme switch with all elements properly styled
- 📜 **Scroll Reveal Animations** — Sections fade & slide up on scroll using IntersectionObserver
- 📱 **Fully Responsive** — Works across desktop, tablet, and mobile
- ⚡ **Zero Dependencies** — No React, no Tailwind, no external JS libraries

## 🚀 Getting Started

### Option 1 — Open directly
Just download and open `index.html` in your browser. No build step needed.

### Option 2 — Clone the repo
```bash
git clone https://github.com/Abdulkalam143/Interactive-3D-portfolio.git
cd Interactive-3D-portfolio
# Open index.html in your browser
```

> **Note:** For best results, open with a local server (e.g. VS Code Live Server extension) to load Google Fonts properly.
